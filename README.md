# react-native-azure-ad-auth

Authentication through Azure AD for React Native

## Installation

```sh
npm install react-native-azure-ad-auth
```

## Usage

```js
import { multiply } from 'react-native-azure-ad-auth';

// ...

const result = await multiply(3, 7);
```
